# Code Of Conduct

## Our Standard

This project is a small open-source package. Contributors are expected to keep
discussion direct, respectful, and focused on improving the software.

Good participation means:

- Use concrete evidence when reporting bugs or reviewing changes.
- Critique ideas and code, not people.
- Keep issues and pull requests on topic.
- Respect privacy. Do not post secrets, private conversation logs, or personal
  information.

Unacceptable behavior includes harassment, threats, sustained off-topic
disruption, or publishing private information without permission.

## Enforcement

Maintainers may edit, hide, or remove comments and may block users who violate
this standard. If you need to report a conduct issue, use the repository issue
tracker or contact the maintainer through the project's GitHub profile.
